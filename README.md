# Simulation système solaire
Simulation de l'interaction entre les planètes et de la courbure de l'espace temps dans le cadre du projet d'année de 3ème bachelier en sciences informatiques à l'ULB


# TMP tracking
To start the project you might want to use `uv` since this is what I used.

Start with:
- `uv sync`
then:
- `uv venv` -> `source .venv/bin/activate`
To run the program, you might want to position yourself in the root folder. After what you can do:
- `uv run python -m src.main`
Enjoy!
